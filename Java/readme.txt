This package contains the the beta version of the java CloudBackend SDK.

Documentation around usage of the sdk is provided in documeation.

lib contains the jar files needed for installing and running the SDK.

Simple_test provides example code that can be run.

testdata is used by the simple test.

This has been tested on Ubuntu LTS 20.04 with OpenJDK 11.
